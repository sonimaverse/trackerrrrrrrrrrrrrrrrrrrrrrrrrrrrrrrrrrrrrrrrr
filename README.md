# Shishir Bhusal's Life OS

A personal second-brain system with real-time sync, diaries, finance tracking, workout & study logging, habit tracking and productivity coaching.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your API key
3. Run the app:
   `npm run dev`
